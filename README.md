Controle de Estoque - Loja de Roupas

Este é um script desenvolvido em Python para otimizar o gerenciamento de inventário em lojas de varejo, com foco inicial na contagem e controle de peças como calças. O objetivo é substituir controles manuais por uma solução automatizada, rápida e de fácil execução via terminal.

🚀 Motivação
Em lojas de roupas, a conferência manual de estoque pode ser demorada e sujeita a erros. Este programa foi criado para facilitar a rotina dos lojistas, permitindo saber exatamente quantas unidades de cada item estão disponíveis em poucos segundos.

🛠️ Funcionalidades
Consulta de Quantidade: Visualização imediata do volume disponível em estoque.

Interface Simplificada: Interação direta via linha de comando (CLI), tornando o uso leve e acessível.

💻 Tecnologias Utilizadas
Linguagem: Python 3.x

Conceitos Aplicados: Manipulação de variáveis, estruturas de repetição e lógica de programação.
